Контрольные точки

Установка контрольных точек

Установка контрольных точек позволяет изменять данные сразу нескольких продуктов. Также есть возможность изменять данные по заказам на завод и продуктам  выбранных продуктов.

Например, для установки у заказа продукта значения реквизита "CDD поставщик":

  • добавьте в справочник "НСИ"-"Логистика"-"Контрольные точки бизнес-процессов" контрольную точку с наименованием "CDD поставщик" (объект "Scan_ЗаказыНаЗавод", реквизит объекта "CDDПоставщик", номер контрольной точки - любой уникальный);

  • перейдите в "Администрирование"-"Сервис"-"Настройки АРМ" и установите флаг у созданной контрольной точки "CDD поставщик" для "АРМ отдела логистики";

  • перейдите в "АРМ Отдел логистики";

  • нажмите кнопку "Контрольные точки";

  • в списке продуктов двойным кликом по первой колонке выберите несколько продуктов, для которых нужно установить контрольную точку;

  • для контрольной точки "CDD поставщик" укажите значение даты;

  • нажмите кнопку "ОК".

В списке контрольных точек можно одновременно устанавливать значения сразу для нескольких контрольных точек.

Особенности использования

"Логистический тип"

При установке значения контрольной точки "Логистический тип" (объект "Scan_Продукты", реквизит объекта "ТипПродуктаЛогистический"), будут выполнены следующие действия:

  • установка значения реквизита "Тип (логистический)" справочника "Продукты";
  • установка значения реквизита "Не обновлять логистические типы" справочника "Продукты".

"Модель продукта"

При установке значения контрольной точки "Модель продукта" (объект "Scan_Продукты", реквизит объекта "МодельПродукта"), будут выполнены следующие действия:

  • установка значения реквизита "Модель продукта" справочника "Продукты";
  • установка значения реквизита "Не обновлять модели" справочника "Продукты";
  • установка значения реквизита "Модель продукта" справочника "Продукты";
  • установка значения реквизита "Не обновлять модели" справочника "Продукты".

"Продукт в архиве"

Для установки контрольной точки для выбранных продуктов должны выполняться следующие условия:

  • "Дата продажи продукта" должна быть заполнена;
  • "Дата передачи продукта дилеру" должна быть заполнена;
  • не должно быть замечаний по изделию продукта, либо "Крайняя дата устранения недостатков" должна быть меньше текущей даты.

"Есть букинг-лист"

При установке контрольной точки отображается специальная вспомогательная форма, на которой необходимо указать данные букинг-листа:"№ букинг-листа", "Дата", а также для автоматическо формирования перемещения продукта необходимо заполнить поля "Место доставки" и "CDD поставщик".

После установки контрольной точки автоматически отправляется письмо о загрузке букинг-листа. 

"Получен грузовой билет"

При установке контрольной точки отображается специальная вспомогательная форма, на которой необходимо указать данные букинг-листа:"№ грузового билета", "Дата грузового билета". После установки контрольной точки автоматически отправляется письмо о получении грузового билета.
