Справочник "Шаблоны согласования"

Предназначен для быстрого заполнения нового бизнес-процесса "Согласование" (команда "Заполнить по шаблону" меню "Все действия" в карточке бизнес-процесса).

Реквизиты

  • Наименование шаблона;
  • Наименование - наименование бизнес-процесса, создаваемого по данному шаблону;
  • Описание;
  • Срок;
  • Важность;
  • Список согласующих лиц - выбирается из справочника пользователей;
  • Вариант согласования (всем сразу или по очереди).

Поля "Наименование шаблона", "Ответственный" и "Комментарий" носят информационно-справочный характер и не влияют на создание новых бизнес-процессов по этому шаблону.

В списке адресатов для согласования (внизу формы) можно указать одного или несколько исполнителей. С помощью упорядочивания списка можно установить последовательность процедуры согласования. А с помощью переключателя "Всем сразу" и "По очереди" - назначить способ рассылки согласования. Если выбрано "По очереди", то после согласования текущим исполнителем, задача переходит на согласование следующему по списку исполнителю.

Способ вызова

Вызывается из раздела "Администрирование" -> "Визирование документов (уат) -> Шаблоны согласования"
